
# on-stop

  Execute when stop typing.

## Installation

  Install with [component(1)](http://component.io):

    $ component install yuehu/on-stop

## API

```js
var stop = require('on-stop');

stop(el, function() {
    console.log('stop');
})
```

### stop(el, func, delay)

Default delay time is 400ms.
